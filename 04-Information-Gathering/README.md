# Project 04 - Information Gathering

## Objective

The objective of this project is to perform passive information gathering using common Linux networking and reconnaissance tools. The project demonstrates how to collect publicly available information about a target domain without performing intrusive activities.

---

## Tools Used

- Kali Linux
- whois
- dig
- host
- nslookup
- traceroute
- curl

---

## Commands Performed

- whois example.com
- dig example.com
- host example.com
- nslookup google.com
- curl -I https://github.com
- traceroute example.com

---

## Skills Learned

- Passive Reconnaissance
- WHOIS Lookup
- DNS Enumeration
- IP Address Resolution
- HTTP Header Analysis
- Route Tracing
- Basic OSINT Techniques
